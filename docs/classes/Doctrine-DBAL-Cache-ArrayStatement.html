public fetch([mixed $fetchMode = null ][, mixed $cursorOrientation = PDO::FETCH_ORI_NEXT ], mixed $cursorOffset) : mixed
Parameters
$fetchMode : mixed = null

Controls how the next row will be returned to the caller. The value must be one of the constants, defaulting to .

$cursorOrientation : mixed = PDO::FETCH_ORI_NEXT

For a ResultStatement object representing a scrollable cursor, this value determines which row will be returned to the caller. This value must be one of the \PDO::FETCH_ORI_* constants, defaulting to \PDO::FETCH_ORI_NEXT. To request a scrollable cursor for your ResultStatement object, you must set the \PDO::ATTR_CURSOR attribute to \PDO::CURSOR_SCROLL when you prepare the SQL statement with \PDO::prepare().

$cursorOffset : mixed

For a ResultStatement object representing a scrollable cursor for which the cursorOrientation parameter is set to \PDO::FETCH_ORI_ABS, this value specifies the absolute number of the row in the result set that shall be fetched. For a ResultStatement object representing a scrollable cursor for which the cursorOrientation parameter is set to \PDO::FETCH_ORI_REL, this value specifies the row to fetch relative to the cursor position before ResultStatement::fetch() was called.
